# Chat Application using TCP and UDP

## TCP Chat

### On Linux
 - Open new terminal
 - `chmod +x TCPChat.sh`
 - `./TCPChat.sh`

### On Windows
 - Open new terminal
 - `javac TCPServer.java && javac TCPClient.java`
 - `java TCPServer`
 - Open 2 new terminals
 - run `java TCPClient` on both


 ## UDP Chat

### On Linux
 - Open new terminal
 - `chmod +x UDPChat.sh`
 - `./UDPChat.sh`

### On Windows
 - Open new terminal
 - `javac UDPServer.java && javac UDPClient.java`
 - `java UDPServer`
 - Open new terminal
 - run `java UDPClient`
